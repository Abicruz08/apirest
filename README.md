### Intrucciones para ejecutar proyecto desde consola
```
Clonar el proyecto del repositorio GitHub
git clone https://github.com/Abicruz08/apirest.git.
Abrir nuestro proyecto con intellij y descargar las dependencias del proyecto.
Abrir una terminal desde nuestro proyecto raiz.
Ejecutar el comando: mvn clean package
En la misma terminar acceder a la carpeta Target.
Ejecutar el archivo .jar con el siguiente comando.
java -jar api-0.0.1-SNAPSHOT.jar
```
